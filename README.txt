Descripción del Proyecto
El proyecto trata de la creación de una maqueta de una app para entrenar dentro de videojuegos, esto en base a distintas rutinas que te puedan ayudar a mejorar dentro del mismo juego. 

Para ello se creo una maqueta basada en que los usuarios puedan crear sus rutinas y difundirlas con otros usuarios, basando una puntuación de las rutinas en base a la comunidad de los propios usuarios.


Requerimientos Funcionales
-Poder iniciar sesión.
-Poder buscar rutinas de entrenamiento.
-Poder escoger el juego del que se quiere buscar rutinas.
-Poder crear tus propias rutinas de entrenamiento.
-La App te da recomendaciones de rutinas.
-Tener alguna forma de feedback para ver que tan popular o buena es la rutina.

Requerimientos No Funcionales
-Debe poder guardar variedad de usuarios/contraseñas.
-Los usuarios deben poder guardar gran cantidad de rutinas en sus librerías.
-Debe haber un link agenciado a cada rutina, para que los usuarios las puedan compartir.
